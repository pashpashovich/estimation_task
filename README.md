# Task

Web-based user registration system

## Content

- [Stack](#stack)

## Stack

- Spring boot
- Hibernate
- Postgres
- Flyway

## How to run

1. Clone or download repository.
2. Start Docker.
3. Execute the following command in the root directory of project:

```
docker compose up --build
```

4. You can go to localhost:8080 and start working.
